PHP File Upload Size Limit Error

How to fix the "POST Content-Length exceeds the limit" error

When uploading files in PHP, you might encounter the following error:

Warning: POST Content-Length of 44380073 bytes exceeds the limit of 8388608 bytes in Unknown on line 0

This error occurs when you try to upload a file that exceeds PHP's configured maximum upload size.

How to Fix This Error

You need to increase PHP's file upload limits by modifying your PHP configuration. Here are the main directives you need to change:

; Maximum allowed size for uploaded files
upload_max_filesize = 64M

; Must be greater than or equal to upload_max_filesize
post_max_size = 64M

; Maximum execution time for each script (in seconds)
max_execution_time = 300

; Maximum input time for each script (in seconds)
max_input_time = 300

; How many files can be uploaded at once
max_file_uploads = 20

Where to Make These Changes

Important: The method for changing these settings depends on your server environment and whether you have access to the main PHP configuration.

Option 1: Modify php.ini (Recommended)

Find your php.ini file (often located in /etc/php/ or your PHP installation directory) and update the values:

; Locate these lines in your php.ini and change them:
upload_max_filesize = 64M
post_max_size = 64M
max_execution_time = 300
max_input_time = 300

Option 2: Use .htaccess (Apache only)

If you're using Apache and have AllowOverride Options privileges, you can add these lines to your .htaccess file:

php_value upload_max_filesize 64M
php_value post_max_size 64M
php_value max_execution_time 300
php_value max_input_time 300

Option 3: Use ini_set() in PHP (Limited effectiveness)

For some settings, you can use ini_set() in your PHP script, but note that upload_max_filesize and post_max_size can only be set in php.ini or .htaccess:

// Place this at the top of your PHP script
ini_set('max_execution_time', 300);
ini_set('max_input_time', 300);
ini_set('memory_limit', '128M');

Steps to Apply Changes

  1. Locate your php.ini file (you can use phpinfo() to find its location)
  2. Edit the values for upload_max_filesize, post_max_size, and time limits
  3. Save the changes
  4. Restart your web server (Apache, Nginx, etc.) for the changes to take effect
  5. Verify the changes using phpinfo() or a simple PHP script that outputs ini_get('upload_max_filesize')
